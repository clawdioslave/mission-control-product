## 📸 SCREENSHOT BEST PRACTICES

### How to Capture Perfect Screenshots:

**Mac:**
1. Open Mission Control in Chrome
2. Press CMD+Shift+4 → Space → Click window
3. Or use CleanShot X (paid, worth it)

**Windows:**
1. Press Win+Shift+S
2. Select area
3. Paste into editor

**Chrome DevTools (Best for precision):**
1. Open Mission Control
2. Right-click → Inspect
3. CMD+Shift+P (Mac) or Ctrl+Shift+P (Windows)
4. Type "screenshot"
5. Select "Capture full size screenshot" or "Capture node screenshot"

### Screenshot Editing:
- Crop to relevant area
- Blur sensitive data (if any)
- Add annotations (arrows, text)
- Consistent padding

## 🎨 DESIGN PRINCIPLES

### 1. Contrast
- Dark background → Light text
- Light background → Dark text
- High contrast for readability

### 2. Hierarchy
- Biggest text = Most important
- Use size, color, weight to show importance
- One main message per graphic

### 3. White Space
- Don't overcrowd
- Breathing room around elements
- Focus attention

### 4. Consistency
- Use same fonts across assets
- Same color palette
- Same photo style

### 5. Mobile-First
- Most people see graphics on phones
- Text must be readable at small sizes
- Test on mobile before publishing
