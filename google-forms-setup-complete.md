## 🚀 SETUP INSTRUCTIONS

### Step 1: Create Day 2 Survey
1. Go to forms.google.com
2. Click "+ Blank"
3. Copy title and description from above
4. Add questions 1-6 (copy/paste)
5. Click "Settings" (gear icon):
   - ✅ Collect email addresses
   - ✅ Show progress bar
6. Click "Customize Theme" (palette icon):
   - Header color: #7b2cbf
   - Background: Light
7. Click "Send" → Click link icon → Click "Shorten URL"
8. Copy the shortened link

### Step 2: Create Day 5 Survey
1. Click "+" to create new form
2. Repeat steps 3-8 from above
3. Add questions 1-15

### Step 3: Enable Response Collection
1. In each form, click "Responses" tab
2. Click green Sheets icon
3. "Create a new spreadsheet"
4. Name it "Mission Control Beta Responses"
